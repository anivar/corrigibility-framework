# Community Assessments

This directory contains corrigibility assessments contributed by the community.

## Structure

```
assessments/
├── [system-name]-assessment.json
├── [system-name]-assessment.json
└── ...
```

## Submitting an Assessment

1. Use the `corrigibility.json` schema
2. Name file: `[system-name]-assessment.json`
3. Include evidence for each test
4. Submit pull request

## Guidelines

- Be specific about what was evaluated and when
- Cite sources (documentation, RTI responses, audits)
- Note limitations and what could not be verified
- Disclose any conflicts of interest

## Multiple Assessments

Different assessors may reach different conclusions. Multiple assessments of the same system are permitted and encouraged.
